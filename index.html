Current Progress

So far, we have implemented the basic physics of fluid particles (position, velocity, and acceleration) with Euler integration, and we have also added collisions between particles. In each time step, we update the velocity of each particle to be v = v + TIMESTEP * a, and the position to be x = x + TIMESTEP * v. We also adapted the code from our cloth simulation project for collisions between particles by readjusting the positions of particles if they get too close to each other.
